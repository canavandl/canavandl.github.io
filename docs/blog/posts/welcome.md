## About This Site

I've rebuilt my personal site using [MkDocs Material](https://squidfunk.github.io/mkdocs-material/), which provides:

- Beautiful, responsive design
- Built-in blog functionality with RSS feed
- Dark mode support
- Fast search
- Easy content management with Markdown

## Getting Started

To add new blog posts:

1. Create a new `.md` file in `docs/blog/posts/`
2. Add frontmatter with date, authors, and categories
3. Write your content in Markdown
4. Deploy with `uv run mkdocs gh-deploy`
