# TxFoster RegRAG Demo

[foster-rag](https://github.com/canavandl/foster-rag) is a retrieval-augmented generation system built to make Texas foster care regulations more accessible to caseworkers and families. The demo below shows the system answering natural language questions grounded in the Texas Administrative Code.
